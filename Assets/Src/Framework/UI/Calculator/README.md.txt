# Calculator Button : 장지원
- 계산기의 숫자 버튼(NumButton), 소수점 버튼(ControllButton 중 일부)
- 버튼을 클릭하면 Result화면에 띄울 수 있도록 이벤트를 전송

# Calculator Clear : 장지원
- clear 버튼을 구현
- C 버튼을 누를 경우 ResultValue 스크립트의 ClearDisplay를 호출

# Calculator Ok : 장지원
- Ok 버튼을 누르면 계산이 완료
- CalculatorManager에 입력한 결괏값을 string으로 보냄

# Calculator Veiw Point : 장지원
- 계산기 클릭시 player 카메라 -> 계산기 고정 카메라 로 이동한다.

# Payment Manager : 장지원
- 구매성공 구매 실패 로직
